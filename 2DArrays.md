## Two Dimensional Arrasy

```java
String [][] cars = [3][3];

cars[0][0] = "Mustang";
cars[0][1] = "F-150";
cars[0][2] = "Mach-E";
cars[1][0] = "CTSV";
cars[1][1] = "ATSV";
cars[1][2] = "2.0T";
cars[2][0] = "Corvette";
cars[2][1] = "Camaro";
cars[2][2] = "Z28";
```


or  

```java
String [][] cars = {
                      {"Mustang","F-150","Mach-E"},
                      {"CTSV","ATSV","2.0T"},
                      {"Corvette","Camaro","z28"},
                    };  
```

* Different ways to to set up 2D or Multidimentional Arrays
